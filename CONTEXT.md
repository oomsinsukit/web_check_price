# Plush Price Checker

เว็บสำหรับนักสะสม/ผู้ขายชาวไทย: ถ่ายรูปตุ๊กตาญี่ปุ่นมือ 2 แล้วได้ "ราคาที่ขายได้จริง" จากตลาดญี่ปุ่น เพื่อใช้ตัดสินใจซื้อเข้าหรือตั้งราคาขาย

## Language

**Plush (ตุ๊กตาผ้า)**:
สินค้าในขอบเขตของระบบ — ตุ๊กตาผ้า/นุยกุรุมิ/มาสคอตจากญี่ปุ่น รวมของตู้คีบ (プライズ)
_Avoid_: ฟิกเกอร์ (อยู่นอกขอบเขต MVP), ของสะสมทั่วไป

**Market Price (ราคาตลาด)**:
ราคาที่สินค้า*ปิดการขายแล้วจริง* คำนวณจาก Sold Listings บน Mercari JP เท่านั้น
_Avoid_: ราคาตั้งขาย, asking price, ราคาป้าย

**Sold Listing**:
ประกาศบน Mercari JP ที่ขายสำเร็จแล้ว — หน่วยหลักฐานราคาหนึ่งชิ้น (รูป, ราคา, วันที่ขาย)
_Avoid_: listing เฉยๆ (กำกวมว่ารวมของที่ยังไม่ขายไหม)

**Identification (การระบุตัวสินค้า)**:
การแปลงรูปถ่ายเป็นชุด keyword ภาษาญี่ปุ่น (ตัวละคร, ประเภท, ไซส์, ผู้ผลิต) เพื่อใช้ค้นหา ไม่ใช่การเทียบรูปต่อรูป
_Avoid_: image matching, visual search

**Hint**:
ข้อความเสริมที่ผู้ใช้พิมพ์ประกอบรูป (เช่น "ได้จากตู้คีบ") — optional เสมอ

**Tiered Result**:
หลักการแสดงผล: มั่นใจสูง → ยืนยันราคาของสินค้าชิ้นนั้น / มั่นใจต่ำ → แสดง Sold Listings ที่คล้ายให้ผู้ใช้เลือกเอง ระบบไม่เดามั่วเป็นตัวเลขเดียวเด็ดขาด

**Match Confirmation (การจิ้มเลือก)**:
การที่ผู้ใช้เลือก Sold Listing ที่ตรงกับของในมือ — ขั้นตอนที่ทำให้ราคาสรุปแม่นระดับ "รุ่นนี้เป๊ะๆ" โดยไม่พึ่ง AI

**Price Summary (สรุปราคา)**:
ช่วงราคา + ค่ากลางของ Sold Listings ที่เกี่ยวข้อง แสดงทั้ง JPY และ THB พร้อมวันที่ขายกำกับ

**Daily Cap (เพดานรายวัน)**:
จำนวนครั้งสูงสุดที่ระบบยอมเรียก Vision LLM ต่อวัน — กลไกล็อกเพดานค่าใช้จ่าย
